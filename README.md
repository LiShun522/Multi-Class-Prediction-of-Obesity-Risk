# 肥胖預測 LINEBOT

基於機器學習和Google Gemini的肥胖預測服務，透過LINE聊天機器人提供互動介面。

## 專案架構
- 機器學習模型用於肥胖風險評估
- Google Gemini API提供智能對話能力
- 部署在Google Cloud Run上的伺服器
- LINE Messaging API實現用戶互動
